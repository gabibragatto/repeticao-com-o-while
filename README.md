---



###  O que eu treinei aqui



Aprendi e pratiquei principalmente:

-  Estruturas de repetição (`while`)



---



##  **Exercícios**



### 1. Número positivo

O programa pede um número e só aceita valores positivos.  

Se for negativo, ele continua pedindo até vir um válido.



---



### 2. Média de notas

O usuário digita várias notas de 0 a 10.  

Quando digita um número negativo, o programa para e mostra a média das notas.



---



### 3. Total de compras

O usuário vai inserindo valores de compras.  

Quando digita **0**, o sistema encerra e mostra o total final.



---



### 4. Nome e sobrenome

O programa pede nome e sobrenome separadamente.  

Se algum estiver vazio, ele pede novamente até ser preenchido corretamente.



---



### 5. Sistema de saque

Simula um caixa eletrônico com saldo inicial.  

O usuário faz saques enquanto houver saldo disponível.  

Se o valor for inválido ou maior que o saldo, ele precisa tentar novamente.  

O sistema encerra quando o saldo acaba.
